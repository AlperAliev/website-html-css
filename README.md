# website-html-css

Omnifood – Healthy Meal Subscription Website

Omnifood is a fully responsive front-end web project that presents a subscription-based healthy meal delivery service. The site allows users to explore subscription plans, learn how the service works, browse featured meals, read customer testimonials, and sign up through a call-to-action form.

Features
1. Header Navigation

Sticky navigation with smooth scrolling.

Links to all major sections:

How it works

Meals

Testimonials

Pricing

Try for free

2. Main Hero Section

Explains the value proposition of the service.

Includes a strong CTA (Call to Action) encouraging users to start a subscription.

3. “How It Works” Section

Step-by-step explanation of using the service.

Includes phone mockup images paired with text descriptions.

4. Meals Section

Showcases two sample meals.

Shows:

User ratings

Calories

Compatible diets (e.g., vegan, gluten-free)

5. Testimonials & Gallery

Four customer testimonials with photos.

Image gallery showcasing healthy meals and happy customers.

6. Pricing Section

Two subscription tiers:

Basic Plan (1 meal/day)

Premium Plan (2 meals/day)

Includes a list of benefits for each plan.

7. Call To Action (CTA) Section

Signup form including:

Full name

Email address

“Where did you hear from us?” dropdown

“Sign up now” button

8. Footer

Contact information

Account section (Login, Create Account, App downloads)

Company section (About, Careers, For Business, etc.)

Resources section (Help Center, Privacy & Terms, Recipe Directory)

Some buttons scroll the page back to top (using JavaScript)

JavaScript Functionality

Smooth scrolling for internal navigation links.

Scroll-to-top behavior for inactive/demo buttons.

Basic UI interactions to improve user experience.

Responsive Design

The website is fully responsive and optimized for:

Mobile phones

Tablets

Laptops

Desktop screens

The layout adapts using modern CSS techniques (flexbox, grid, responsive units, and media queries).

Technologies Used

HTML5

CSS3 (Flexbox, Grid, Media Queries)

JavaScript (ES6+)

Project Purpose

This project demonstrates core front-end development skills including:

Building a responsive website from scratch

Applying clean and reusable CSS architecture

Implementing basic JavaScript interactions

Structuring content for a real-world product webpage
